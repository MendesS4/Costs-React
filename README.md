# Costs

This project is a cost management application built using ReactJS. It helps users track expenses and manage costs for various projects.

## Installation

1. Clone the repository.
2. Install dependencies using `npm install`.
3. Run the application using `npm start`.

## Features

- Add new projects and their associated costs.
- View a summary of total expenses.
- Edit or delete existing projects and expenses.

## Usage

1. Navigate to the project directory.
2. Run the application using `npm start`.
3. Open the application in your web browser.

## Technologies Used

- ReactJS
- HTML/CSS
